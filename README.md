# Attendance Monitoring System Dashboard

A full-stack attendance monitoring platform with SMS notifications for students and faculty.

## Features

- Student attendance tracking
- Faculty dashboard
- Department management
- Attendance reports
- SMS alerts using Twilio
- Authentication system

## Tech Stack

### Frontend
- React.js
- Vite
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- Twilio

## Installation

### Backend

```bash
cd Backend
npm install
npm start
